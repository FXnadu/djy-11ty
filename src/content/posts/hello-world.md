欢迎来到 DJY 站点！这是使用 [Eleventy](https://www.11ty.dev/) 搭建的静态站点。

## 技术栈

- **框架**：Eleventy v3
- **模板**：Nunjucks + Markdown
- **构建**：Node.js

## 快速开始

```bash
npm install
npm start
```

访问 `http://localhost:8080` 查看站点。
